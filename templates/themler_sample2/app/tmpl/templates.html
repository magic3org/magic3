Default Template is disabled.

The Default Template is used for all components which have no appropriate templates (Post, Page, Blog, etc). For example, Joomla Category List Layout, Login page, etc.

Also the Default Template will be applied to the pages created by the external components except VirtueMart pages which have own Page Templates.

To turn on the Default Template, simply create a Menu Item of the chosen Type that has no appropriate Page Template.

Note: The menu item should be accessible from the website Front-End.

Page Template is disabled.

The Page Template is used for Single Article Layouts. The feature of this template is that the article does not have any metadata (author, date, print, email, etc.) by default.

To turn on the Page Template, please follow these steps:

  1. Create an Article: in Joomla Content >> Article Manager choose Add New Article.
    • Add Article Title, Category, content.
    • Publish the Article.
  2. Once you have created the Article you need to link it to the Menu of your website: in Joomla Menus create a Menu or choose an existing one and click on Add New Menu Item.
    • Specify the "Single Article" menu item type.
    • Select the Article you have created.
    • Hide all the Article Metadata under the Article Options.
    • Publish the item.

Note: The menu item should be accessible from the website Front-End.

Blog Template is disabled.

The Blog Template is used for Category Blog and Featured Articles Layouts.

To turn on the Blog Template please follow these steps:

  1. Create a Category with Article(s) in Joomla Content.
    • Create a Category in Joomla >> Content >> Category Manager >> Add New Category
    • Save the Category.
    • Create an Article(s) in Joomla Content >> Article Manager >> Add New Article.
    • Add Article Title, Category, content.
    • Mark the Article as Featured if needed.
    • Publish the Article.
  2. Once you have created a Category with Articles you need to link it to the Menu of your website: in Joomla Menus create a Menu or choose an existing one and click on Add New Menu Item.
    • Specify the "Category Blog" or "Featured Articles" menu item type.
    • Choose the Category.
    • Publish the item.

Note: The menu item should be accessible from the website Front-End.

Post Template is disabled.

The Post Template is used for Single Article Layouts. The feature of this template is that the article should have some article metadata (author, date, print, email, etc).

To turn on the Post Template please follow these steps:

  1. Create an Article: in Joomla Content >> Article Manager choose Add New Article.
    • Add Article Title, Category, content.
    • Publish the Article.
  2. Once you have created the Article you need to link it to the Menu of your website: in Joomla Menus create a Menu or choose an existing one and click on Add New Menu Item.
    • Specify the "Single Article" menu item type.
    • Select the Article you have created.
    • Show any of the article metadata under the Article Options
    • Publish the item.

Note: The menu item should be accessible from the website Front-End.

Shopping Cart Template is disabled.

The Shopping Cart Template is used for VirtueMart Shopping Cart.

To turn on the Shopping Cart Template you need to create a link to it in the Menu of your website: in Joomla Menus create Menu or choose the existing one and click on Add New Menu Item.

  • Specify the "VirtueMart Shopping Cart" menu item type.
  • Publish the Item.

Note: The menu item should be accessible from the website Front-End.

Products Template is disabled.

The Products Template is used for VirtueMart Category Layout.

To turn on the Products Template please follow these steps:

  1. Create a Product Category with Products in Joomla >> Components >> VirtueMart.
    • Create a Product Category in Joomla >> Components >> VirtueMart >> Product Categories.
    • Save the Category.
    • Create Product(s) in Joomla >> Components >> VirtueMart >> Products.
    • Add Product Name, Category, Price, etc.
    • Publish the Product.
  2. Once you have created a Product Category with Products you need to link it to the Menu of your website: in Joomla Menus create a Menu or choose an existing one and click on Add New Menu Item.
    • Specify the "VirtueMart Category Layout" menu item type.
    • Choose the Category.
    • Publish the Item.

Note: The menu item should be accessible from the website Front-End.

Product Details Template is disabled.

The Product Details Template is used for VirtueMart Product Details Layout.

To turn on the Product Details Template please follow these steps:

    • Add Product Name, Price, etc.
    • Publish the Product.
  2. Once you have created the Product you need to link it to the Menu of your website: in Joomla Menus create a Menu or choose an existing one and click on Add New Menu Item.
    • Specify the "VirtueMart Product Details Layout" menu item type.
    • Choose the Product.
    • Publish the Item.

Note: The menu item should be accessible from the website Front-End.

VirtueMart Page Templates are disabled.

There can be two reasons:

  1. You are working in Themler Cloud and chose a theme without VirtueMart.

    To enable VirtueMart Page Template please access Joomla Admin Panel >> Extensions >> Extensions Manager >> Manage and enable VirtueMart Components and Modules.

  2. You are working in your CMS and you do not have VirtueMart extension installed.

To install VirtueMart please follow these steps:

  1. Download the latest stable version here.

  2. Unzip the downloaded file com_virtuemart.2.x.y.extract_first.zip

    This will extract two more zip-files:

    • com_virtuemart.2.x.y.zip - which holds the main VirtueMart component.
    • com_virtuemart.2.x.y_ext_aio.zip - which holds all modules and plugins essential for VirtueMart.
  3. Log in to your Joomla Admin Panel and go to the Extensions Manager.
  4. First, install the com_virtuemart.2.x.y.zip extension.
  5. Second, install com_virtuemart.2.x.y_ext_aio.zip extension.
